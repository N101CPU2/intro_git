# 売上データ分析（Python / SQL）

## 概要
Excelで作成した売上データを用いて、
商品別・顧客別・月別の売上分析を行いました。
データ前処理から集計、可視化、考察までを一貫して実施しています。

## 使用データ
- sales.csv：売上トランザクションデータ
- customers.csv：顧客マスタ
- products.csv：商品マスタ

（※ いずれも学習用に作成した架空データ）

## 使用技術
- Python
  - pandas
  - matplotlib
- SQL（pandasでSQL相当の処理を実装）
- Excel（データ作成）

## 分析内容
1. データの確認・前処理
   - データ型の確認
   - 欠損値の補完・除外
2. 商品別売上分析
   - 商品マスタとJOIN
   - 売上合計を集計
3. 月別売上推移分析
   - 日付を月単位に変換
   - 月別売上を集計
4. 可視化
   - 商品別売上（棒グラフ）
   - 月別売上推移（折れ線グラフ）

## 結果・考察
- 売上は特定の商品に集中していることが分かった
- 月別に売上の増減があり、時期要因の影響が示唆される
- 主力商品への依存低減や販売時期を考慮した施策が有効と考えられる

## 今後の改善点
- 顧客属性（年齢・地域）別の分析
- データ量を増やした検証
- SQLのみでの再実装
  
## SQLによる分析
SQLiteを利用し、CSVデータをデータベースに取り込みました。
PythonからSQLを実行し、JOIN・GROUP BY・WHEREを用いた
売上分析を行っています。

- 商品別売上分析（JOIN + 集計）
- 月別売上分析（日付処理）
- 条件指定による分析（WHERE句）

SQLで取得した結果をpandasのDataFrameとして受け取り、
可視化・考察まで実施しました。

# SQLを用いた売上データ分析

売上データをもとに、商品別・顧客別・地域別の売上分析を行った。

## 使用データ

- customers：顧客マスタ
- products：商品マスタ

## 使用技術
- Python（pandas, matplotlib）
- SQLite
- SQL（JOIN, GROUP BY, WHERE）

## 分析内容
- 商品別売上分析
- 顧客別売上分析
- 地域別売上分析

## 分析結果
- 売上は一部の商品・顧客に集中している傾向が見られた
